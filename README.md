# 🚀 Node.js CI/CD Pipeline with GitHub Actions

## 📌 Project Overview
This project demonstrates how to deploy a Node.js application using a CI/CD pipeline with GitHub Actions.

## ⚙️ Tech Stack
- Node.js
- GitHub Actions
- AWS (EC2 / S3 / Elastic Beanstalk)
- Git & GitHub

## 🔄 CI/CD Workflow
1. Developer pushes code to GitHub
2. GitHub Actions triggers workflow
3. Application is built and tested
4. Deployment to cloud server is automated

## 📁 Project Structure
- app.js
- package.json
- .github/workflows/deploy.yml

## 🚀 How It Works
- On every push to `main` branch:
  - Install dependencies
  - Run build/test
  - Deploy to server
